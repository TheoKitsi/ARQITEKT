# US-1.1 — Product Search & Browse

As a shopper, I want to search and browse products so that I can find items I want to buy.

## Acceptance Criteria
1. Search bar returns results as user types (debounced)
2. Results can be filtered by category, price range, and rating
3. Products display thumbnail, title, price, and rating
4. Empty search shows category-based browsing
