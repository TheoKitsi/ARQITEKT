# US-3.2 — Returns & Refunds

As a customer, I want to request returns so that I can get a refund for unsatisfactory items.

## Acceptance Criteria
1. Return request form with reason selection
2. Return eligibility checked against return policy window
3. Refund processed to original payment method
4. Return status shown in order history
