# US-2.1 — Shopping Cart Management

As a shopper, I want to manage items in my cart so that I can review my order before checkout.

## Acceptance Criteria
1. Items can be added, removed, and quantity updated
2. Cart shows line item subtotals and order total
3. Cart persists across browser sessions (localStorage or server-side)
4. Cart badge in header shows item count
