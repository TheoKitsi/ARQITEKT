# US-4.1 — Customer Registration & Login

As a visitor, I want to create an account so that I can save my information for faster future checkouts.

## Acceptance Criteria
1. Registration with email, password, and name
2. Email verification required before full access
3. Login with email/password or social providers
4. Remember-me option for persistent sessions
