# US-4.2 — Wish List

As a customer, I want to save products to a wish list so that I can purchase them later.

## Acceptance Criteria
1. Heart icon on product cards toggles wish list membership
2. Wish list page shows all saved items with current prices
3. Items can be moved directly from wish list to cart
4. Price change indicator when saved price differs from current
