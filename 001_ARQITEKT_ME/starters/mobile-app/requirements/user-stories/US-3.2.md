# US-3.2 — Push Notifications

As a user, I want to receive push notifications so that I am informed about important updates.

## Acceptance Criteria
1. App requests notification permission on first launch
2. Notifications display title, body, and optional image
3. Tapping a notification opens the relevant screen (deep link)
4. User can manage notification preferences in settings
