# US-1.2 — Profile Management

As a user, I want to view and edit my profile so that I can keep my information up to date.

## Acceptance Criteria
1. User can view profile with avatar, name, and email
2. User can update name and avatar image
3. Changes are saved immediately with optimistic UI
4. Validation errors shown inline on form fields
