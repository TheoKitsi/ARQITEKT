# US-3.1 — Offline Data Access

As a user, I want previously loaded content to be available offline so that I can use the app without internet.

## Acceptance Criteria
1. Viewed items are cached locally for offline access
2. Offline indicator shown when network is unavailable
3. Cached data refreshes automatically when back online
4. User can manually trigger sync from settings
