# US-3.2: Create & Edit Records

As a user, I want to create new records and edit existing ones through a form so that I can manage application data.

## Acceptance Criteria
- [ ] AC-3.2.1: Form validates required fields before submission
- [ ] AC-3.2.2: Success/error feedback shown after save
- [ ] AC-3.2.3: Edit prefills form with existing values
- [ ] AC-3.2.4: Unsaved changes trigger confirmation on navigation
