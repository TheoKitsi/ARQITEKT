# US-1.1: User Registration & Login

As a new user, I want to create an account and log in so that I can access the application securely.

## Acceptance Criteria
- [ ] AC-1.1.1: User can register with email and password
- [ ] AC-1.1.2: User can log in with valid credentials
- [ ] AC-1.1.3: Invalid credentials show clear error message
- [ ] AC-1.1.4: Password meets minimum security requirements (8+ chars, mixed case, digit)
