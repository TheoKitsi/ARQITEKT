# US-2.2: Interactive Charts & Filters

As a user, I want to interact with charts and apply date/category filters so that I can explore data from different perspectives.

## Acceptance Criteria
- [ ] AC-2.2.1: Charts support hover tooltips with data details
- [ ] AC-2.2.2: Date range picker filters all dashboard widgets
- [ ] AC-2.2.3: Category filter narrows data to selected segments
- [ ] AC-2.2.4: Filter state persists during the session
