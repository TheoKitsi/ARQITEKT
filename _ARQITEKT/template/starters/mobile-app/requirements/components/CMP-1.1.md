# CMP-1.1 — Auth Screen

## Responsibility
Login/register screen with email, password, social auth buttons, and biometric toggle.

## Technology
Flutter/React Native screen with secure storage for tokens.

## Interfaces
- Input: user credentials
- Output: auth token stored securely
- API: `POST /api/auth/login`, `POST /api/auth/register`
