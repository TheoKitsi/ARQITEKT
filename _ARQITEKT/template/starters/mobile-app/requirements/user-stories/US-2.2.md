# US-2.2 — Detail Screen & Actions

As a user, I want to view item details and perform actions so that I can interact with content.

## Acceptance Criteria
1. Detail screen shows full item content with images
2. User can perform primary action (like, save, share)
3. Back navigation returns to previous scroll position
4. Loading and error states handled gracefully
