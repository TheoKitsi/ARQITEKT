# US-3.2 — Subscription & Billing

As a tenant admin, I want to manage my subscription plan so that I can control costs and access features.

## Acceptance Criteria
1. Tenant can view current plan and usage metrics
2. Plan upgrade/downgrade takes effect at next billing cycle
3. Usage metering tracks API calls per tenant per month
4. Billing webhooks from payment provider update plan status
