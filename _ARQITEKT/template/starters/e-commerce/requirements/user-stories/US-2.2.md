# US-2.2 — Checkout Flow

As a shopper, I want a streamlined checkout process so that I can complete my purchase quickly.

## Acceptance Criteria
1. Guest checkout available without account creation
2. Steps: shipping address, payment method, order review
3. Address validation with autocomplete suggestions
4. Order summary visible throughout checkout process
