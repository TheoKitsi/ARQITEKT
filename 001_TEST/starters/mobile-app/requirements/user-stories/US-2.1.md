# US-2.1 — Home Screen & Feed

As a user, I want a home screen with a scrollable feed so that I can see the latest content at a glance.

## Acceptance Criteria
1. Home screen shows a scrollable list of content items
2. Pull-to-refresh reloads content from API
3. Skeleton placeholders shown during initial load
4. Tapping an item navigates to its detail screen
