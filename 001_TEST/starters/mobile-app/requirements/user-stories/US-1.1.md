# US-1.1 — Mobile Login & Biometrics

As a mobile user, I want to log in with email/password or biometrics so that I can securely access my account quickly.

## Acceptance Criteria
1. User can log in with email and password
2. User can enable Face ID / fingerprint after first login
3. Failed biometric falls back to password entry
4. Session persists across app restarts until explicit logout
