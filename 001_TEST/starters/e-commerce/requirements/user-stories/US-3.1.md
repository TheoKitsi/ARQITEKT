# US-3.1 — Order Tracking

As a customer, I want to track my order status so that I know when to expect delivery.

## Acceptance Criteria
1. Order status page shows current stage (confirmed, shipped, delivered)
2. Tracking number links to carrier's tracking page
3. Status change triggers email notification
4. Estimated delivery date displayed when available
