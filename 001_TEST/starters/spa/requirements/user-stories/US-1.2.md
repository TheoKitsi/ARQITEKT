# US-1.2: Password Reset & Account Management

As a registered user, I want to reset my password and manage my profile so that I maintain access and keep my information current.

## Acceptance Criteria
- [ ] AC-1.2.1: User can request password reset via email
- [ ] AC-1.2.2: Reset link expires after 24 hours
- [ ] AC-1.2.3: User can update display name and avatar
- [ ] AC-1.2.4: User can change password from settings
