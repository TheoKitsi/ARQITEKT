# US-3.1: Data Table with Search & Pagination

As a user, I want to browse data in a table with search, sorting, and pagination so that I can find and review entries efficiently.

## Acceptance Criteria
- [ ] AC-3.1.1: Table displays data with sortable columns
- [ ] AC-3.1.2: Search bar filters rows by text content
- [ ] AC-3.1.3: Pagination controls navigate through large datasets
- [ ] AC-3.1.4: Column widths are adjustable
