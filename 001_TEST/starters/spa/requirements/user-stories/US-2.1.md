# US-2.1: Dashboard Overview

As a user, I want to see a dashboard overview when I log in so that I can quickly understand key metrics at a glance.

## Acceptance Criteria
- [ ] AC-2.1.1: Dashboard shows KPI summary cards (count, trend, change)
- [ ] AC-2.1.2: Charts render within 1 second
- [ ] AC-2.1.3: Dashboard is responsive on mobile screens
- [ ] AC-2.1.4: Data refreshes automatically or on pull-to-refresh
