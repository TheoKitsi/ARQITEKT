# FN-10.1.1.3: Schwartz Value Prioritization Test

## Specification

The system shall provide an optional value prioritization test based on the Schwartz model (e.g. security, stimulation, hedonism, achievement, power, universalism, benevolence, tradition, conformity, self-direction).
