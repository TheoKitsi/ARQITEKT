# FN-16.1.1.4: Visibility Labeling in Preview

## Specification

The system shall clearly label visibility levels in the profile preview --- lock icon for Level 1 (algorithm only), eye icon for Level 2 (visible in preview) --- so the user immediately recognizes who sees what.
