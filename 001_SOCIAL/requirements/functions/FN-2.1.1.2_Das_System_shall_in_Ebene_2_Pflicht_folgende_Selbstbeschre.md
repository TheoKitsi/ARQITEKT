# FN-2.1.1.2: Level 2 Self-Description Fields

## Specification

The system shall provide the following self-description fields for Level 2 (Mandatory): personality type, introversion/extraversion, communication style, daily routine, diet, smoking/alcohol/drugs, pets, religion/spirituality, political orientation.
