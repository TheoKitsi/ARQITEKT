# FN-16.1.1.2: Default Level Assignment

## Specification

The system shall assign sensible default visibility levels per field type --- income, sexual preferences, kinks, health information = Level 1; age, core values, intention, hobbies, rough region = Level 2; name, exact location, photos, free texts, contact = Level 3.
