# FN-12.1.1.3: Relocation Willingness and Distance

## Specification

The system shall capture willingness to relocate (Yes / Only within X km / No) and maximum distance as mandatory fields in funnel level 3 and use them as matching criteria. -> Fields defined in SOL-2, Level 3.
