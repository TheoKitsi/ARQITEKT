# FN-3.2.1.2: Improvement Hints

## Specification

The system shall give the user specific improvement hints (e.g., "Fill Level 4 for approx. +15% match quality", "Add your hobbies for more accurate proposals"). (Dashboard display see SOL-8, FN-8.3.1.1)
