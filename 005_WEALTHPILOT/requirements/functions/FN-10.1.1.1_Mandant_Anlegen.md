# FN-10.1.1.1: Mandant Anlegen

> **Parent**: [CMP-10.1.1](../components/CMP-10.1.1_Mandanten_Verwaltungsmodul.md)

---

## Functional Description

Das System muss neue Mandanten anlegen: Firmenname, Subdomain, Kontaktperson, Vertragsbeginn, Lizenzmodell (Starter/Professional/Enterprise).

- Das System soll die Subdomain auf Verfuegbarkeit pruefen.
- Das System soll bei Mandanten-Anlage automatisch Default-Konfigurationen erstellen.
- Das System soll den Mandanten nach Anlage sofort lauffaehig machen (Zero-Config-Start).

---

## Preconditions

- Nutzer hat die Rolle "Super-Admin".
- Mandanten-Verwaltungsmodul ist geoeffnet.

---

## Behavior

1. Admin fuellt das Mandanten-Formular aus: Firmenname, Subdomain, Kontaktperson (Name + E-Mail), Vertragsbeginn, Lizenzmodell.
2. System validiert die Subdomain: nur alphanumerische Zeichen + Bindestrich, min 3 Zeichen, verfuegbar (kein Duplikat).
3. System erstellt den Mandanten-Datensatz.
4. System generiert Default-Konfigurationen: Standard-Branding, Standard-Feature-Flags (nach Lizenzmodell), Standard-Produktfreigabe.
5. System erstellt den initialen Admin-Account fuer den Mandanten (E-Mail an Kontaktperson mit Aktivierungslink).
6. System zeigt Erfolgsmeldung mit Mandanten-ID und Subdomain.

---

## Postconditions

- Mandant ist angelegt und konfiguriert.
- Default-Konfigurationen sind erstellt.
- Mandanten-Admin hat Aktivierungslink erhalten.
- Mandant ist unter {subdomain}.wealthpilot.de erreichbar.

---

## Error Handling

- Das System soll bei doppelter Subdomain die Meldung "Subdomain bereits vergeben" anzeigen.
- Das System soll bei fehlendem Pflichtfeld das Feld rot markieren.
- Das System soll bei E-Mail-Versandfehler den Mandanten trotzdem anlegen und den E-Mail-Versand in die Retry-Queue stellen.

---

## Acceptance Criteria (functional)

- [ ] Mandant "Musterbank" mit Subdomain "musterbank" wird angelegt.
- [ ] Doppelte Subdomain wird abgelehnt.
- [ ] Default-Feature-Flags fuer "Starter"-Lizenz sind gesetzt.
- [ ] Aktivierungs-E-Mail wird an Kontaktperson gesendet.
- [ ] Mandant ist unter musterbank.wealthpilot.de erreichbar.

---

## Notifications

- Aktivierungslink per E-Mail an den Mandanten-Admin (NTF-MANDANT-WELCOME).
