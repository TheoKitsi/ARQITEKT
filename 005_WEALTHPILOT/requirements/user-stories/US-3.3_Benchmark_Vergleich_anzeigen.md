# US-3.3: Benchmark Vergleich anzeigen

> **Parent**: [SOL-3](../solutions/SOL-3_Rendite_Engine.md)

---

## User Story

Als Nutzer moechte ich meine Portfolio-Rendite mit relevanten Benchmarks vergleichen, damit ich einschaetzen kann ob meine Strategie marktkonform ist.

---

## Acceptance Criteria

- [ ] AC-US-3.3.1: Auswahl aus Standard-Benchmarks: DAX, MSCI World, MSCI Europe, S und P 500, Inflation
- [ ] AC-US-3.3.2: Overlay-Chart: Eigene Rendite vs. Benchmark ueber gleichen Zeitraum
- [ ] AC-US-3.3.3: Outperformance/Underperformance in Prozentpunkten und absolut anzeigen
- [ ] AC-US-3.3.4: Custom-Benchmark: Nutzer kann eigene Gewichtung erstellen
