# US-4.3: Optimale Umschichtungsstrategie vorschlagen

> **Parent**: [SOL-4](../solutions/SOL-4_Impact_Simulation.md)

---

## User Story

Als Nutzer moechte ich vom System einen optimalen Umschichtungsvorschlag erhalten, damit ich den geringstmoeglichen negativen Impact auf mein Portfolio habe.

---

## Acceptance Criteria

- [ ] AC-US-4.3.1: System generiert bis zu 3 optimierte Szenarien basierend auf Risikoprofil
- [ ] AC-US-4.3.2: Optimierungskriterien: minimaler Rendite-Verlust, maximale Liquiditaetserhaltung, steuerliche Effizienz
- [ ] AC-US-4.3.3: Side-by-side Vergleich aller Szenarien inkl. Impact-Score
- [ ] AC-US-4.3.4: Empfehlung wird mit Begruendung hervorgehoben
- [ ] AC-US-4.3.5: Nutzer kann manuell anpassen und sofort Neuberechnung triggern
- [ ] AC-US-4.3.6: Export des gewaehlten Szenarios als PDF mit allen Details
