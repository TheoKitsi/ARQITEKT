# US-1.2: Einnahmen Ausgaben Profil erstellen

> **Parent**: [SOL-1](../solutions/SOL-1_Finanzprofil_und_Onboarding.md)

---

## User Story

Als Nutzer moechte ich mein monatliches Einnahmen- und Ausgabenprofil erfassen, damit das System meine Sparfaehigkeit und finanzielle Belastbarkeit berechnen kann.

---

## Acceptance Criteria

- [ ] AC-US-1.2.1: Nutzer kann regelmaessige Einnahmen erfassen (Gehalt, Mieteinnahmen, Dividenden, Sonstiges)
- [ ] AC-US-1.2.2: Nutzer kann regelmaessige Ausgaben erfassen (Miete, Versicherungen, Sparplaene, Lebenshaltung)
- [ ] AC-US-1.2.3: System berechnet monatlichen Ueberschuss/Defizit automatisch
- [ ] AC-US-1.2.4: Sparquote wird als Prozentsatz des Nettoeinkommens angezeigt
- [ ] AC-US-1.2.5: Kategorisierung nach fixen und variablen Kosten
