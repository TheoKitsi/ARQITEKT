# US-4.2: Cross Impact berechnen und visualisieren

> **Parent**: [SOL-4](../solutions/SOL-4_Impact_Simulation.md)

---

## User Story

Als Nutzer moechte ich die Auswirkungen meiner geplanten Umschichtung auf mein gesamtes Portfolio sehen, damit ich eine informierte Entscheidung treffen kann.

---

## Acceptance Criteria

- [ ] AC-US-4.2.1: Rendite-Differenz wird pro betroffener Position berechnet (vorher vs. nachher)
- [ ] AC-US-4.2.2: Opportunitaetskosten werden ueber 1/3/5/10 Jahre projiziert
- [ ] AC-US-4.2.3: Impact-Score (-100 bis +100) wird als aggregierter Gesamtkennwert angezeigt
- [ ] AC-US-4.2.4: Steuerliche Auswirkungen der Umschichtung werden separat dargestellt
- [ ] AC-US-4.2.5: Visualisierung als Sankey-Diagramm (Kapitalfluesse) und Waterfall-Chart (Rendite-Aenderungen)
- [ ] AC-US-4.2.6: Auswirkung auf bestehende Sparplaene wird explizit angezeigt
