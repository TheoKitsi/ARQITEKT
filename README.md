# ARQITEKT Workspace

> KI-gestütztes Requirements Engineering mit VS Code Copilot Agent Mode

## Quick Start

**Doppelklick auf `start.cmd`** — startet Dashboard im Hintergrund und öffnet den Browser.

VS Code wird über das Dashboard geöffnet (pro Projekt).

Oder manuell:
```bash
cd _ARQITEKT
npm run dashboard    # → http://localhost:3333
```

## Struktur

| Ordner | Inhalt |
|--------|--------|
| `_ARQITEKT/` | Framework-Hub: Dashboard, Projekt-Template |
| `001_SOCIAL/` | Erstes Projekt: SOCIAL Dating Platform |

## Neues Projekt

1. Dashboard öffnen → "Neues Projekt" klicken
2. Name eingeben → Erstellen
3. "In VS Code öffnen" → Copilot Agents sofort verfügbar
